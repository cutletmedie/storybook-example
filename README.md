## Запуск
1. Установить [Node.js](https://nodejs.org/en/download/) >= 22.3.0 или через [nvm](https://github.com/nvm-sh/nvm)

2. Установить зависимости
```sh
npm i
```

3. Запустить проект
```sh
npm start
```

4. (опционально) Запустить Storybook
```sh
npm run storybook
```
